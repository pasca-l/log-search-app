# Flutter project


## Developing
1. Start docker container at repository root.
```bash
$ make start
```

2. Install `Dev Container` VSCode extension, and "reopen in container".

3. Run flutter app inside the `Dev Container`
```bash
$ flutter run -d web-server
```
